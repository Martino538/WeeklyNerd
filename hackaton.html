Home Hackathon

Context: Tijdens een hackathon georganiseerd door de minor Webdesign and Development werden we opgedeeld in groepjes. Onze taak was om in een week tijd een website te maken voor CSS Day, een jaarlijks evenement met sprekers die alles over de nieuwste snufjes rondom CSS vertellen. We besloten om een line-up te maken voor het evenement en hadden de beschikking over een JSON-bestand met informatie over de sprekers. Ik verdiepte me bewust in het fetchen van deze data en het uitlezen naar de browser, omdat ik dit nog niet eerder had gedaan. Ja, bij het vak Webapps From Scratch. Maar bij dit vak was dit een stuk geavanceerder. Iets wat ik hieronder verder ga toelichten. Hoewel ik veel leerde, ontstonden er problemen door inconsistente code conventies binnen het team, wat leidde tot stress en een hoge werkdruk.

Wat heb ik hiervan geleerd?

  • Werken met gefetchte data: Deze hackathon bood me de gelegenheid om geavanceerdere technieken toe te passen als het gaat om data fetchen. Voor elke spreker moest namelijk een kaartje worden gegenereerd met elke kaart de eigenschappen van de bijbehorende spreker. Ik heb geleerd om te werken met appendchild, een for-loop en toggle tussen CSS-classes.

  • Belang van Code Conventies: Hoewel we als team afspraken om code conventies aan te houden, hielden niet alle teamleden zich hieraan. Dit leidde tot problemen bij het implementeren van hun code, wat uiteindelijk resulteerde in een verhoogde werkdruk voor mij.

  • Stressmanagement: Door de last-minute problemen en hoge werkdruk werd ik genoodzaakt om met een stressvolle situatie om te gaan.
