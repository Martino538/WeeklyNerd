Home Nerd 4 - Nils Binder

Grid staat boven flexbox!

Datum: 06-03-2024

Spreker: Nils Binder

Korte samenvatting

Nils Binder is hoofd webdevelopment bij 9 Elements. Aan zijn lezing te horen had hij veel kennis over CSS. Naast dat hij onze gastspreker was, is hij ook onze gastdocent geweest. Hij heeft mij geholpen bij het aanleren van de :has selector tijdens het vak CSS To The Rescue. Door zijn uitleg heb ik destijds veel nieuwe inzichten gekregen!

Aantekeningen

Terug naar zijn lezing... Ik heb een aantal aantekeningen gemaakt over wat hij vertelde tijdens zijn lezing.

  • Het container element bepaalt vaak de maximale breedte en geeft soms padding mee. Soms centreerd het ook de content
  • Er zijn drie mogelijke manier om een container te stylen:
    1. Wrapper { Max-width: 75rem; Margin: 0 auto; Padding: 0 1.5rem; }
    2. Wrapper { Width: min(100% - 3rem, 75rem); Margin-inline: auto; }
    3. Wrapper { Margin-inline: max(1.5rem, ((100% - 75em) / 2)); }

  • Hiernaast heeft hij ook nog een eigen kolommen structuur bedacht.

    .div { display: grid; Grid-template-columns: 2fr 3fr auto 1fr; } H1 { Grid-columns: 2 / 4;} P {grid-columns: 2 / 4;}

Reflectie

Nils Binder vond ik een boeiende spreker, omdat hij veel kennis in CSS heeft. Zijn manier van positionering vond ik ook erg pakkend, maar momenteel weet ik nog niet hoe ik grid moet gebruiken. Dit zou ik mijzelf eerst moeten aanleren. Ook vond ik het ondenkbaar dat er zoveel verschillende soorten units bestaan. Echter denk ik dat ik voor nu zijn gedeelde kennis niet zo snel gebruiken, omdat het vrij complex is en vooral gericht op positionering in tijdschriften.

Hij heeft mij wel aan het denken gezet. Momenteel gebruik ik eigenlijk alleen flexbox, maar ik heb het idee dat je met grid meer controle hebt. Ik wil daarom mijzelf grid ook in de toekomst gaan aanleren.
